## Contributing

Thanks for using and improving this starter!

### Getting started

1. Fork and clone the repo
2. Install deps: `npm install`
3. Run dev: `npm run dev` (http://localhost:3000)
4. Build: `npm run build && npm start`

### Making changes

- Keep the starter neutral and template-friendly
- Update `README.md` if behavior changes
- Run quality checks:
  - `npm run lint`
  - `npm run format:check`
  - `npm test` (placeholder)

### Submitting PRs

1. Open an issue first for large changes
2. Use the PR template and describe motivation and testing
3. Keep diffs focused and small when possible

### Release workflow

- Tag stable versions (e.g., `v1.0.0`)
- Consider creating a GitHub release with notes
